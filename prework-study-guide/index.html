HTML

  • The head element contains information about the webpage, but most will not be visible.
  • The body element represents the visible content shown to ther user
  • The body and head are known as parent elements. Child elements are found contained within their respective parent element.
  • HTML is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how muc space we want around the content inside an element.
  • The class selector has to proceed property assignments from the CSS; you can also select multiple class selctors or even name new ones to apply CSS.
  • * = applies CSS properties to everything.

Git

  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • To add changes to current working branch: stage->commit->pull down changes from base branch->push changes
  • git add -A: adds motifications in the current working branch to staging area. Note: -A indicates you want to add all changes
  • git commit -m "any comments": commit with message
  • git pull origin main: double checks our local branch is in sync with the base branch in GitHub. Note: Should get message it is already up to date.
  • git push origin branch-name: pushes changes to the remote branch.
  • Check you saved it correctly; update local branch (git checkout main), then pull the remote branch onto the local branch (git pull origin main)

JavaScript

  • console.log(var-name): allows you to see value of variable with Chrome Devtools.
